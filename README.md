# Smart Money Calc & Scientific Calculator

A professional dual-purpose application built with **React Native** and **Expo**.

## 🚀 Live Demo
Check out the web version here: [vaibhavs-smart-calculator.surge.sh](https://vaibhavs-smart-calculator.surge.sh)

## ✨ Features
* **Money Counter:** Quickly calculate total cash amounts by entering bill/coin counts.
* **Scientific Calculator:** Full mathematical functions with a sleek dark-themed UI.
* **Responsive Design:** Optimized for both mobile and web screens.

## 🛠️ Tech Stack
* React Native / Expo
* TypeScript
* Vector Icons
